# scheduleKTM

Simple Web Crawler for KTMB's E-Ticketing System

## Functionality

+ Quick ticket availability retrieval
+ Minimalistic Web User Interface & Telegram Bot

## Usage

+ Replace your own Telegram API Password in __telegram_bot.py__
+ Run __main.py__
